# Requirements Documentation

This folder contains all requirements and specification documents for the Laundry App project.

## 📄 Documents

### [SRS.md](SRS.md) - Software Requirements Specification
Complete specification document including:
- Functional requirements
- Non-functional requirements
- System features
- Interface requirements
- Design constraints

## How to Use

### Adding Your SRS Document
1. Edit [SRS.md](SRS.md) with your requirements
2. Or replace it with your existing SRS document
3. Keep the file named `SRS.md` or use `SRS.pdf` if you have a PDF version

### Supported Formats
- **Markdown (.md)** - Best for version control and GitHub viewing
- **PDF (.pdf)** - For formal submissions
- **Word (.docx)** - Can be added to this folder

## Additional Documents

You can add other requirements-related documents to this folder:
- Use case diagrams
- User stories
- Acceptance criteria
- Test requirements
- System architecture diagrams
