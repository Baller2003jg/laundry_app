## Hosting Options

Once you deploy your app, update the demo link above. Here are some popular hosting options:

### Free Hosting Services:
- **[Netlify](https://www.netlify.com/)** - Best for React apps, free SSL
- **[Vercel](https://vercel.com/)** - Optimized for Vite projects
- **[GitHub Pages](https://pages.github.com/)** - Free hosting through GitHub
- **[Render](https://render.com/)** - Free tier with automatic deploys
- **[Cloudflare Pages](https://pages.cloudflare.com/)** - Fast global CDN

### Quick Deploy Instructions

#### Deploy to Netlify (Recommended)
1. Build your app: `npm run build`
2. Drag and drop the `dist/` folder to [Netlify Drop](https://app.netlify.com/drop)
3. Get your live URL instantly
4. Update the link above!

#### Deploy to Vercel
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

#### Deploy to GitHub Pages
```bash
# Install gh-pages
npm install --save-dev gh-pages

# Add to package.json scripts:
# "deploy": "vite build && gh-pages -d dist"

# Deploy
npm run deploy
```
